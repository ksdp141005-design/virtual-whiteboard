# Virtual Whiteboard 🎨

*Interactive digital canvas enabling dynamic drawing, geometric shape rendering, and image exportation.*

### 🚀 Live Demo
[Click here to view the live application](https://virtual-whiteboard-w4ld-pxbtz8s7w-ksdp141005-designs-projects.vercel.app/)

### 🛠️ Tech Stack
* **Frontend:** React, Next.js, HTML5 Canvas
* **Styling:** Tailwind CSS, CSS Modules
* **State Management:** React Context API, `useReducer`
* **Deployment:** Vercel

### ✨ Key Features
* **Dynamic Rendering Engine:** Utilizes HTML5 Canvas and Rough.js for freehand and geometric drawing.
* **Custom Toolbox UI:** Independent state control for brush size, stroke colors, and fill colors.
* **Smart Export Pipeline:** Downloads the active canvas as a high-quality PNG with a forced solid-white background to prevent transparency rendering issues.
* **Robust Undo/Redo:** Custom history stack for safe and instantaneous state reversals without memory leaks.
* **Native Grid System:** Interactive CSS-free dot-grid background generation.
